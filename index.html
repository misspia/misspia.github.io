    • projects.
        • D3 Audio Visualizer
        • Watch music come to life. This visualizer dynamically colors and creates animated graphs of uploaded audio files. Graphs are generated by utilizing frequency and time domain data passed through custom made D3 components and then into the DOM as SVG elements. Animations are done by repeatedly drawing graphs in new animation frames and colors are changed by varying rgb values with audio data. App currently includes 2 color themes (warm, cool) and 3 graph types (radial, frequency, wave).
        • Technology: D3.js, JavaScript, Sass, HTML5, Gulp
        • React 3D League of Legends Champion Selector
        • A compilation of LoL Champion data and images scraped using Node.js and rendered with React.js. The unique interface utilizes 3D transform properties in CSS to give the selection pannel the illusion of perspective. Images are optimized with the use of gulp.
        • Technology: React.js, Node.js, Sass, HTML5, Gulp
        • AngularJS Pokedex
        • A highly interactive recreation of the classic digital Pokemon encyclopedia. I wrote a scrapper to collect the Pokemon data and mass download sprites and images. The interface is mobile compatible and was built using AngularJS. Responsive layout utilizes the CSS flexbox module.
        • Technology: AngularJS, Node.js, Sass, HTML5, Gulp
